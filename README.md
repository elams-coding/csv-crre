# CSV-CRRE

Ceci est un projet console **Java** qui permet l'affichage du contenu d'un fichier **CSV**.

---

## Projet

Ce projet a été créé pour la création et la lecture de fichiers CSV depuis la console.

Il est exécutable sur toutes machines (Windows, Linux, MacOS) disposant de la version Java 21.0.9 ou version ultérieure.

Pour le moment le projet est simplement un fichier java. Un jar sera produit plus tard lorsque le projet sera terminé.
